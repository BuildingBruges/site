We want to get together with software developers in Bruges, so we can get to know each other and learn from each other. We want to play our part in making Bruges an attractive location for developers and companies.  
Anyone with an interest in software development is welcome. We will be focussing on subjects that matter for professional software developers, but all skill levels can join.  
We want to organize meetups, talks, workshops, get-togethers, drinks, etc.
